前端部分主要用来展示数据，将数据可视化为图表，并管理好页面布局方便使用者明确的得到各个部分的详细数据并且能轻松的看出各个措施下各个变量的对比
主要用到的技术栈有：HTML、CSS、JavaScript、Less、ECHARTS、Jquery、Ajax
后端部分主要用来接受前端发送的请求，并处理各种数据和进行状态机的实现，同时将各部分更新完的数据通过ajax发送给前端
后端部分主要基于Springboot和JavaWeb通过Maven提供的一系列依赖完成开发
背景：新冠疫情肆虐下各国采取了不同的防疫措施，得到了不同的防疫效果，我想了解在不同的防疫措施下患病人数、疫情能否得到控制或者疫情是否加重、反弹等情况，并基于模拟得到的数据进行分析，得出各种措施是否能有效降低甚至防止疫情的扩散
